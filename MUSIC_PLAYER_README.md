## 🚀 安装步骤

### 1. 导入数据库表

**方式一：phpMyAdmin导入**
1. 登录 phpMyAdmin
2. 选择你的数据库（默认是 `lovey`）
3. 点击"导入"
4. 选择 `migration_add_music.sql` 文件
5. 点击"执行"

**方式二：命令行导入**
```bash
mysql -u root -p lovey < D:/phpStudy/PHPTutorial/WWW/migration_add_music.sql
```

### 2. 创建API目录
确保 `api` 目录存在：
```bash
mkdir -p D:/phpStudy/PHPTutorial/WWW/api
```

### 3. 访问后台管理
1. 登录后台：`http://your-domain/admin/`
2. 在左侧菜单找到"音乐播放器"
3. 点击"添加音乐"按钮

### 4. 添加音乐
填写以下信息：
- **歌曲名称**（必填）：音乐标题
- **艺术家**：歌手名称
- **音乐文件URL**（必填）：音乐文件的完整URL
  - 支持格式：mp3, m4a, ogg
  - 示例：`https://example.com/music.mp3`
- **封面图URL**：封面图片的完整URL
  - 留空则使用默认封面
  - 示例：`https://example.com/cover.jpg`
- **时长(秒)**：歌曲时长（可选）
- **排序**：数字越小越靠前
- **启用**：是否在前台显示

## 🎵 音乐文件来源建议

### 推荐方案：
1. **使用CDN托管**
   - 上传到阿里云OSS、腾讯云COS
   - 使用七牛云、又拍云等CDN服务

2. **使用音乐API**
   - 网易云音乐API（需处理版权）
   - QQ音乐API（需处理版权）

3. **本地托管**
   - 在服务器创建 `music` 目录
   - 上传音乐文件
   - URL示例：`http://your-domain/music/song.mp3`

### 注意事项：
- ⚠️ 请确保音乐文件有合法使用权
- ⚠️ 注意音乐文件大小，建议单个文件不超过10MB
- ⚠️ 使用HTTPS协议可避免混合内容问题

## 🐛 常见问题

### Q: 播放器不显示？
A: 检查 `footer.php` 是否正确引入了CSS和JS文件

### Q: 音乐无法播放？
A: 
1. 检查音乐URL是否正确
2. 检查音乐文件是否支持（mp3最佳）
3. 检查浏览器控制台是否有CORS错误

### Q: 显示"暂无音乐"？
A: 
1. 确认已导入数据库表
2. 确认已添加音乐并启用
3. 检查 `api/getMusicList.php` 文件权限

### Q: 后台菜单没有"音乐播放器"？
A: 清除浏览器缓存，重新登录后台
