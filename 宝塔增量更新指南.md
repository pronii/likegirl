### 方法三：逐个文件替换（最安全）

#### 在宝塔面板文件管理器中：

1. **更新 head.php**
   - 找到 `head.php` → 右键 **编辑**
   - 在 `<link rel="shortcut icon"` 这行**之前**添加：

```html
<!-- ===== DNS Prefetch + Preconnect 优化（解决 SSL 握手延迟） ===== -->
<!-- DNS 预解析：提前解析域名，节省 DNS 查询时间 -->
<link rel="dns-prefetch" href="//fonts.googleapis.com">
<link rel="dns-prefetch" href="//cdn.jsdelivr.net">
<link rel="dns-prefetch" href="//fonts.gstatic.com">

<!-- 预连接：提前建立 TCP + TLS 连接，节省 233ms SSL 握手时间 -->
<link rel="preconnect" href="https://fonts.googleapis.com" crossorigin>
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link rel="preconnect" href="https://cdn.jsdelivr.net" crossorigin>
```

   - 修改 Google Fonts 引用（合并请求）：
     找到这三行：
     ```html
     <link href="https://fonts.googleapis.com/css?family=Concert+One|Pacifico" rel="stylesheet">
     <link href="https://fonts.googleapis.com/css2?family=Noto+Serif+SC:wght@700&display=swap" rel="stylesheet">
     <link href="https://fonts.googleapis.com/css2?family=Noto+Serif+SC:wght@400&display=swap" rel="stylesheet">
     ```
     
     替换为：

     <link href="https://fonts.googleapis.com/css?family=Concert+One|Pacifico&display=swap" rel="stylesheet">

## ⚙️ 宝塔 Nginx 配置更新（核心优化）

### 步骤：

1. 宝塔面板 → **网站** → 找到你的站点 → **设置**
2. 左侧 **配置文件** → 在 `server {}` 块内添加：

```nginx
server {
    # 原有配置...
    
    # ===== 性能优化配置（新增） =====
    
    # 静态资源强缓存（图片1年）
    location ~* \.(png|jpg|jpeg|gif|webp|ico|svg)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
        add_header X-Cache-Status "HIT-STATIC";
        access_log off;
    }

    # CSS/JS 缓存 30 天
    location ~* \.(css|js)$ {
        expires 30d;
        add_header Cache-Control "public";
        access_log off;
    }

    # 字体文件缓存 1 年
    location ~* \.(woff|woff2|ttf|eot|otf)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
        access_log off;
    }
    
    # 原有的 location / 和 location ~ \.php$ 配置保持不变...
}
```

3. 点击 **保存**
4. 点击 **重载配置**

## ⚠️ 注意事项

### 1. 备份原文件
在修改任何文件前，先在宝塔面板右键 → **复制** → 重命名为 `.bak` 备份

### 2. 清除缓存
更新完成后：
- 浏览器：`Ctrl+Shift+Delete` 清除缓存
- 宝塔：如果安装了 Redis/Memcached，清除缓存

### 3. 不要覆盖数据库配置
**不要更新** `admin/Config_DB.php`，保留线上的数据库配置！

### 4. 不要覆盖上传目录
**不要更新** `uploads/` 目录，保留线上的图片数据！

## 🐛 常见问题

### Q1: 更新后图片不显示？
**解决**：检查 `uploads` 目录权限
```bash
chmod -R 777 /www/wwwroot/你的站点/uploads
```

### Q2: 懒加载不生效？
**解决**：
1. 检查 `lazyload.js` 是否上传成功
2. 检查 `loveImg.php` 是否正确引入脚本
3. 清除浏览器缓存

### Q3: Nginx 配置报错？
**解决**：检查语法，每个配置块后面要有分号 `;`
