## 🎯 核心功能模块

### 1️⃣ 前台功能（用户端）

| 文件名 | 功能 | 说明 |
|--------|------|------|
| `index.php` | 首页 | 展示情侣头像、恋爱计时器、功能卡片 |
| `little.php` | 点点滴滴 | 文章/日记列表页 |
| `page.php` | 文章详情 | 单篇文章/日记详情展示 |
| `leaving.php` | 留言板 | 访客留言，自动获取QQ头像 |
| `loveImg.php` | 恋爱相册 | 图片展示，支持灯箱预览 |
| `list.php` | 恋爱列表 | 情侣约定事件清单 |
| `about.php` | 关于我们 | BotUI机器人对话式展示 |

### 2️⃣ 后台功能（管理端）

**位置**: `admin/` 目录

| 功能模块 | 相关文件 |
|---------|---------|
| 登录认证 | `login.php` |
| 后台首页 | `index.php` |
| 基础设置 | 情侣昵称、网站标题、背景图等 |
| 文章管理 | `articleAdd.php`, `articleMod.php`, `articleDel.php` |
| 留言管理 | `leavingSet.php`, 违禁词设置 |
| 相册管理 | `albumSet.php`, 批量操作、转移相册 |
| 恋爱清单 | 事件列表管理 |
| 关于设置 | `aboutSet.php` - 配置BotUI对话 |
| 安全设置 | 密码修改、安全码 |
| 自定义 | 自定义CSS、页头页脚代码 |
| IP管理 | IP黑名单 |

### 3️⃣ API接口

| 文件名 | 功能 |
|--------|------|
| `getAlbums.php` | 获取相册列表 |
| `getPhotos.php` | 获取照片数据（返回id、图片路径等） |
| `transferPhotos.php` | 照片批量转移 |
| `admin/ajax.php` | 后台Ajax接口 |

## 🗄️ 数据库结构

### 主要数据表

| 表名 | 说明 |
|------|------|
| `text` | 网站基本信息（标题、情侣昵称、背景图、恋爱开始时间） |
| `article` | 文章/日记内容 |
| `leaving` | 留言数据 |
| `lovelist` | 恋爱事件列表 |
| `loveImg` | 恋爱相册图片 |
| `about` | 关于页面配置（BotUI对话内容） |
| `login` | 管理员登录信息 |
| `diySet` | 自定义设置（CSS、页头页脚代码） |
| `leavSet` | 留言设置（违禁词、字数限制） |
| `IPerror` | IP黑名单 |
| `warning` | 安全日志 |

### 数据库迁移文件

- `love_db.sql` - 完整数据库初始化
- `database_update_all.sql` - 全量更新
- `migration_*.sql` - 增量更新脚本（相册、音乐、缩略图等）

## 🎨 前端技术栈

### UI组件库

- **BotUI** - 机器人对话界面（关于页面）
- **Toastr** - 弹窗提示
- **NProgress** - 页面加载进度条
- **Spotlight.js** - 图片灯箱
- **FunLazy** - 图片懒加载
- **Animate.css** - CSS动画
- **jQuery Pjax** - 无刷新页面切换

### 样式结构

- 全局样式：`Style/css/`
- 后台样式：`admin/assets/css/`
- 自定义字体图标：`Style/Font/`

## 🚀 部署相关

### 部署脚本

- `bt_deploy.sh` - 宝塔面板部署脚本
- `restart_nginx.bat` - Windows Nginx重启
- `restart_nginx_tls.bat` - TLS配置Nginx重启
- `create_update_package.bat` - 创建增量更新包

### 环境要求

- PHP >= 7.4
- MySQL >= 5.7
- Apache/Nginx
- 目录需要读写权限
