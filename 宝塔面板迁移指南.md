# 🚀 LikeGirl 迁移到宝塔面板完整指南

## 📋 环境要求

### 宝塔面板最低配置
- **PHP**: 7.4+ (推荐 8.0+)
- **MySQL**: 5.6+ (推荐 8.0+)
- **Nginx**: 1.18+ 或 Apache 2.4+
- **磁盘空间**: 至少 500MB

---

## 🎯 迁移步骤

### 第一步：宝塔面板准备

#### 1.1 创建站点
1. 登录宝塔面板 (http://你的服务器IP:8888)
2. 左侧菜单点击 **网站** → **添加站点**
3. 填写信息：
   ```
   域名: yourdomain.com (或直接用IP)
   根目录: /www/wwwroot/likegirl
   PHP版本: 7.4 或 8.0
   数据库: MySQL (创建同名数据库)
   ```
4. 点击 **提交**

#### 1.2 创建数据库
如果上一步没有自动创建：
1. 左侧菜单 **数据库** → **添加数据库**
2. 填写：
   ```
   数据库名: likegirl
   用户名: likegirl
   密码: 设置一个强密码（记住！）
   ```

---

### 第二步：上传代码文件

#### 方案A：Git 克隆（推荐）

**在宝塔终端执行**：
```bash
# 1. 进入站点根目录
cd /www/wwwroot/likegirl

# 2. 克隆仓库（会提示输入GitHub凭据）
git clone https://github.com/pronii/likegirl.git .

# 3. 设置权限
chown -R www:www /www/wwwroot/likegirl
chmod -R 755 /www/wwwroot/likegirl

# 4. uploads 目录需要写权限
chmod -R 777 /www/wwwroot/likegirl/uploads
```

#### 方案B：手动上传

1. 打包本地代码：
   ```bash
   # Windows本地执行
   cd D:\phpStudy\PHPTutorial\WWW
   tar -czf likegirl.tar.gz --exclude=.git --exclude=ip.txt *
   ```

2. 宝塔面板上传：
   - 左侧 **文件** → 进入 `/www/wwwroot/likegirl`
   - 点击 **上传** → 选择 `likegirl.tar.gz`
   - 上传完成后右键 → **解压**

3. 设置权限（在宝塔终端）：
   ```bash
   cd /www/wwwroot/likegirl
   chown -R www:www .
   chmod -R 755 .
   chmod -R 777 uploads
   ```

---

### 第三步：导入数据库

#### 3.1 导出本地数据库

**在本地 phpStudy 执行**：

```bash
# Windows 本地 CMD/PowerShell
cd D:\phpStudy\PHPTutorial\MySQL\bin

# 导出数据库（输入密码：root）
mysqldump.exe -u root -p likegirl > D:\likegirl_backup.sql
```

#### 3.2 导入到宝塔数据库

**方案A：宝塔面板导入**（推荐，文件 < 50MB）

1. 宝塔面板 → **数据库** → 找到 `likegirl` 数据库
2. 点击 **管理** → 进入 phpMyAdmin
3. 左侧选择 `likegirl` 数据库
4. 顶部 **导入** → 选择文件 → 上传 `likegirl_backup.sql`
5. 点击 **执行**

**方案B：命令行导入**（文件较大时）

1. 上传 SQL 文件到服务器：
   ```bash
   # 使用宝塔文件管理器上传到 /tmp/likegirl_backup.sql
   ```

2. 在宝塔终端执行：
   ```bash
   mysql -u likegirl -p likegirl < /tmp/likegirl_backup.sql
   # 输入你设置的数据库密码
   ```

---

### 第四步：修改数据库配置

**编辑 `admin/Config_DB.php`**：

1. 宝塔面板 → **文件** → 进入 `/www/wwwroot/likegirl/admin/`
2. 右键 `Config_DB.php` → **编辑**
3. 修改以下内容：

```php
<?php
// 数据库地址（宝塔默认）
$db_address = "localhost";

// 数据库用户名（改为你创建的）
$db_username = "likegirl";

// 数据库密码（改为你设置的强密码）
$db_password = "你的数据库密码";

// 数据库名
$db_name = "likegirl";
```

4. **保存文件**

---

### 第五步：配置 Nginx（关键！）

#### 5.1 应用性能优化配置

1. 宝塔面板 → **网站** → 找到你的站点 → **设置**
2. 左侧 **配置文件** → 在 `server {}` 块内添加：

```nginx
server {
    listen 80;
    server_name yourdomain.com;
    root /www/wwwroot/likegirl;
    index index.php index.html;

    # ===== 性能优化配置（重要！） =====
    
    # 静态资源强缓存（图片1年）
    location ~* \.(png|jpg|jpeg|gif|webp|ico|svg)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
        add_header X-Cache-Status "HIT-STATIC";
        access_log off;
    }

    # CSS/JS 缓存 30 天
    location ~* \.(css|js)$ {
        expires 30d;
        add_header Cache-Control "public";
        access_log off;
    }

    # 字体文件缓存 1 年
    location ~* \.(woff|woff2|ttf|eot|otf)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
        access_log off;
    }

    # PHP 处理
    location ~ \.php$ {
        fastcgi_pass unix:/tmp/php-cgi-80.sock;  # 根据你的PHP版本调整
        fastcgi_index index.php;
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
        include fastcgi_params;
    }

    # 禁止访问隐藏文件
    location ~ /\. {
        deny all;
    }

    # 禁止访问备份文件
    location ~* \.(bak|backup|old|sql)$ {
        deny all;
    }
}
```

3. 点击 **保存** → **重载配置**

#### 5.2 修改 Nginx 主配置（HTTP Keep-Alive 优化）

1. 宝塔面板 → **软件商店** → **Nginx** → **设置**
2. **配置修改** → 找到 `http {}` 块，确保包含：

```nginx
http {
    # 已有的配置...
    
    # HTTP Keep-Alive 优化
    keepalive_timeout 65;
    keepalive_requests 100;
    tcp_nopush on;
    tcp_nodelay on;

    # Gzip 压缩优化
    gzip on;
    gzip_min_length 1k;
    gzip_buffers 4 32k;
    gzip_comp_level 6;
    gzip_types text/plain text/css application/json application/javascript 
               application/xml text/javascript image/svg+xml;
    gzip_vary on;
    gzip_disable "MSIE [1-6]\.";

    # 其他配置...
}
```

3. **保存** → **重载配置**

### 第六步：设置文件权限

在宝塔终端执行：

```bash
cd /www/wwwroot/likegirl

# 设置所有者
chown -R www:www .

# 基础权限
find . -type d -exec chmod 755 {} \;
find . -type f -exec chmod 644 {} \;

# uploads 目录需要写权限
chmod -R 777 uploads

# admin 目录写权限（用于图片上传）
chmod -R 755 admin
```

---

### 第七步：PHP 配置优化

1. 宝塔面板 → **软件商店** → **PHP 7.4/8.0** → **设置**
2. **配置修改** → 修改以下参数：

```ini
; 上传文件大小限制（用于上传照片）
upload_max_filesize = 100M
post_max_size = 100M

; 脚本执行时间
max_execution_time = 300
max_input_time = 300

; 内存限制
memory_limit = 256M

; 时区设置
date.timezone = Asia/Shanghai
```

3. **保存** → **重启 PHP**

#### 7.2 安装必要的 PHP 扩展

在 **PHP 设置** → **安装扩展** 中，确保已安装：
- ✅ mysqli

- ✅ fileinfo

## 🔒 安全加固（重要！）

### 1. 修改后台路径（可选）

```bash
cd /www/wwwroot/likegirl
mv admin admin_secure_2026
```

然后访问：`http://你的域名/admin_secure_2026/`

### 2. 禁止目录浏览

宝塔站点设置 → **配置文件** → 确保有：
```nginx
location / {
    autoindex off;  # 禁止目录浏览
}
```

### 3. 防火墙规则

宝塔面板 → **安全** → 添加规则：
- 放行 80 端口（HTTP）
- 放行 443 端口（HTTPS，如果配置了SSL）
- 限制 3306 端口（MySQL）仅本地访问

### 4. 定期备份

宝塔面板 → **计划任务** → **添加任务**：
```
任务类型: 备份数据库
执行周期: 每天 凌晨3点
数据库: likegirl
保留份数: 7
```

## 🐛 常见问题排查

### Q1: 访问显示 404

**解决**：
1. 检查站点根目录是否正确
2. 检查 Nginx 配置中的 `root` 路径
3. 确认文件已上传完整

### Q2: 显示 500 错误

**解决**：
```bash
# 查看 PHP 错误日志
tail -f /www/wwwroot/likegirl/error.log

# 或查看 Nginx 错误日志
tail -f /www/server/nginx/logs/error.log
```

### Q3: 数据库连接失败

**解决**：
1. 检查 `admin/Config_DB.php` 配置是否正确
2. 测试数据库连接：
   ```bash
   mysql -u likegirl -p -h localhost likegirl
   ```
3. 确认数据库用户有权限

### Q4: 图片上传失败

**解决**：
```bash
# 检查 uploads 目录权限
ls -la /www/wwwroot/likegirl/uploads

# 如果权限不够
chmod -R 777 /www/wwwroot/likegirl/uploads
```

### Q5: 性能优化不生效

**解决**：
1. 宝塔面板 → Nginx → 重载配置
2. 清除浏览器缓存（Ctrl+Shift+Delete）
3. 使用 Chrome F12 Network 面板验证
