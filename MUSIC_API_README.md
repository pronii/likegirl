# 音乐播放器 - API版安装说明

## 🎵 功能特性

使用**网易云音乐API**，无需本地存储音乐文件：
- ✅ 搜索全网音乐，一键添加
- ✅ 自动获取封面、歌手、时长
- ✅ 实时播放，无需上传文件
- ✅ 支持数百万首正版音乐

## 🚀 快速开始

### 第一步：导入数据库

**首先执行原始表创建**（如果还没执行）：
```sql
-- 在 phpMyAdmin 导入 migration_add_music.sql
```

**然后执行API支持更新**：
```sql
-- 在 phpMyAdmin 导入 migration_update_music_api.sql
```

或命令行：
```bash
mysql -u root -p lovey < D:/phpStudy/PHPTutorial/WWW/migration_add_music.sql
mysql -u root -p lovey < D:/phpStudy/PHPTutorial/WWW/migration_update_music_api.sql
```

### 第二步：使用音乐搜索功能

1. 登录后台 `http://your-domain/admin/`
2. 点击左侧 **🎵 音乐播放器**
3. 点击 **搜索添加音乐** 按钮
4. 输入歌曲名或歌手名，如：`告白气球`
5. 点击搜索结果中的任意歌曲即可添加
6. 访问前台页面，右下角自动出现播放器

## 📁 文件说明

### 新增/修改的文件：

1. **migration_update_music_api.sql** - 数据库表更新（添加平台和song_id字段）
2. **api/musicApi.php** - 网易云音乐API代理（搜索、获取详情、播放地址）

5. **api/getMusicList.php** - 前端获取音乐列表（自动生成播放地址）

### 原有文件保持不变：
- 播放器CSS和JS无需修改
- 其他API文件保持不变

## 🔧 API接口说明

### 搜索音乐
```
GET /api/musicApi.php?action=search&keyword=歌曲名
```

返回：
```json
{
  "success": true,
  "data": [
    {
      "id": "186016",
      "name": "告白气球",
      "artist": "周杰伦",
      "album": "周杰伦的床边故事",
      "cover": "http://p1.music.126.net/...",
      "duration": 223
    }
  ]
}
```

### 获取播放地址
```
GET /api/musicApi.php?action=url&id=186016
```

返回：
```json
{
  "success": true,
  "url": "https://music.163.com/song/media/outer/url?id=186016.mp3"
}
```

## 🐛 常见问题

### Q: 搜索不到结果？
A: 
1. 检查服务器能否访问 `music.163.com`
2. 检查PHP是否开启cURL扩展
3. 尝试更换搜索关键词

### Q: 添加后无法播放？
A: 
1. 部分歌曲可能有版权限制
2. 检查浏览器控制台是否有跨域错误
3. 尝试添加其他歌曲测试

### Q: 播放器不显示？
A: 确保已在前台页面引入CSS和JS（footer.php已包含）

### Q: 想换成QQ音乐API？
A: 修改 `api/musicApi.php` 中的API地址和解析逻辑
